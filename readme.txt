Ziltag brings conversation to all your images.

== Description ==

Ziltag is a visual tagging plugin that helps you discover and discuss wonderful things. By directly connecting users to everything on images, we make information more accessible and shareable.

== Installation ==

1. Sign up for https://ziltag.com
2. Setup your website domain, and you'll get a plugin script with a plugin token.
3. Copy the plugin token to Ziltag Wordpress plugin settings page.

== Frequently Asked Questions ==

= Where to get my Ziltag token? =

You can retrieve your plugin token in [Ziltag Dashboard](https://ziltag.com/dashboard/account),
it's 6 letter long, composed of digits and letters.
